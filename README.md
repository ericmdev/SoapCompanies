## SOAP Companies

A PHP SOAP application distributed in a Debian Docker container.

### Getting Started

Add soap configuration to `/webapp/app/config/parameters.yml`.

Simply download [Docker Quickstart Terminal](https://docs.docker.com/mac/step_one/) and run the following commands:

    make
    make run

After a few seconds, visit: <docker_machine_ip>

See the Makefile for more options.
